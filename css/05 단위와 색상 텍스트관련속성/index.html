  • font-family:글꼴

  • word-spacing: 어간
  • text-align: 글자정렬
  • line-height:행간

unit

  • px : 화면을 구성하는 논리적 단위의최소 단위
  • em : 부모크기의 배율
  • rem : 16px의 배율
