display 출력방식

block 레벨

  • 한줄을 통째로 차지하는 요소 block
  • 기본적으로 부모의 넓이와 동일한 넓이를 가지게됨
  • 크기를 가질 수 있음

inline 레벨

  • 줄내림이 되지 않는 요소
  • 크기속성을 가질 수 없음
  • a, small, em, i, span 등
span

inline-block 레벨

  • 하이브리드 속성, 줄내림이 되지 않으면서 크기를 가질 수 있음
  • padding,margin,width,height 크기 속성을 부여할 수 있음
  • 인라인 블럭 요소 : canvas,svg,iframe,input,img,button,video,input,select,textarea 등을 들 수 있다.
span

inline 레벨간의 수평줄맞춤

span span span
  • text-align은 요소 안쪽 글자 및 인라인계열 요소를 정렬
  • line-height은 행간을 제어하는 속성이지만, 요소의 높이와 동일하게 설정할 경우 수직 중앙정렬 기능으로 사용됨, (!한줄일 때만 사용)
  • vertical-align 속성은 글자 및 인라인계열 요소들의 형제요소들끼리 수평 줄맞춤, 정렬하는 속성
  • margin적용 X
