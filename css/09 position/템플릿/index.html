좌표를 이용해서 특정 위치로 요소를 배치

absolute

  • abs 포지션이 적용되면 독립적인 위치를 가진다 다른요소와 겹치고, 부모는 높이를 인식할 수 없다.
  • 좌표를 설정할 수 있음 : left, right, top, bottom
  • absolute의 좌표 기준점은 position속성을 가지고 있는 부모 기준이다.
box1

fixed속성과 absolute속성의 차이

  • absolute는 문서기준, fixed는 브라우저 기준
  • fixed의 좌표는 부모와 상관없음.
    • (!단 부모요소에 transform속성이 적용되면 예외)

relative

  • absolute가 적용된 자식요소의 기준으로 설정할때 사용.
  • 현재 레이아웃을 그대로 유지하면서 position속성을 적요할 수 있음
