속성선택자

.ex2_1 a[href] : href속성을 가지고있는 a요소

.ex2_1 a[href=http://] : href속성의 값이 http://와 일치하는 a요소

.ex2_1 a[href^='http'] : href속성 값이 http로 시작되는 a요소

.ex2_1 a[href$='net'] : href속성 값이 net으로 끝나는 a요소

.ex2_1 a[href*='.'] : href속성 값에 .을 포함하는a요소
