유투브 동영상 삽입  

1. <video> 요소로 동영상을 넣을수 있지만, 용량문제로 유투브에 올려서 링크하는 방식사용

2. 유투브 사이트 해당 동영상 재생상태에서 오른쪽 버튼 > 소스코드 복사하기 

3. html 문서에서 동영상 파일이 들어갈 영역을 지정한후 그안에 복사된 소스코드 붙여넣기 

4. 웹접근성 에서는 <iframe> 요소를 사용하는 것보다 <embed> 요소로 바꾸는 것이 좋음 

5. 동영상 관련 여러가지 옵션 설정 ( &로 계속 이어서 지정 ) 

      자동재생 :  ?autoplay=1&mute=1 ( 웹접근성 )     
      영상시작시점 설정 : &start=100 (초)
      관련동영상 안보이게 설정 : &rel=0
      다른영상재생 : &playlist=영상아이디       
      반복재생 : &loop=1 (한개 영상으로 계속반복할려면 &playlist 항목에 같은 영상 아이를 지정한다.) 
      상단정보 안보이게 설정  : &showinfo=0 (2018.9 지원종료 ) 
      하단 컨트롤바 안보이게 설정 : &controls=0 
      마우스오버시 컨트롤바 보이게 설정 : &autohide=1
